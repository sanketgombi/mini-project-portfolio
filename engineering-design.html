Conceptual Design

Different design concepts explored for the automated flower sorting system

Concept 01

This system is an automated visual inspection and sorting setup where a camera captures images of items passing on a conveyor belt under controlled LED illumination. Image data is processed to analyze quality and a segregator redirects items into color-specific bins.

Concept 02

An automated color sorting system where objects move from a hopper onto a conveyor. A color sensor identifies each item, and servo-driven wipers divert it into the correct chute. An industrial robot may assist in high-speed handling.

Concept 03

A conveyor-based sorting system using proximity sensors to detect objects. Sensor signals activate solenoid plungers that direct items into designated chutes. This allows efficient real-time classification based on size or color.

Concept 04

A DC motor‚Äìdriven rack and pinion based sorting mechanism. Sensor data determines flower quality, and motor-driven linear motion shifts sorting gates accordingly, ensuring smooth and gentle handling for delicate products.
